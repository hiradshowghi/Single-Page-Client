Install Instructions:

To run the code make sure to have npm installed by doing the following:

npm install 

Also, install extension jQuery:

npm install jquery

Launch instructions:

Once both are downloaded go to the directory containing the zip folder and its files. 

Enter "node server.js" which is the server file name to host the server and press enter. Which you will then see the following:

Server Running at PORT 3000  CNTL-C to quit
To Test
http://localhost:3000/index.html

Copy and paste this link into your browser and you will see the search bar where you are able to search for a song name and if it is available you will be presented with the lyrics and chords.
